### Mini Project
## LLM-based Agent with Telegram Bot for Research Purpose

### Summary
This repo is an archived storage for a mini-project in Natural Language Processing course at the Catholic University of America as a part of the assignment.

### Project Hierachy

Main file: `research-agent.py`

This file is organized into four (4) sections
1. **Mock data creation**: a dictionary of fake and real papers with tittles and abstracts.
2. **Tools for the agents**: a set of four (4) tools implemented by the agent includes searching for relevant papers, summarizing abstracts, extracting keywords, and comparing two papers.
3. **Agent loop**: a looping function to receive, process, and reply to user query
4. **CLI implementation**: a function to run the agent loop locally
5. **Telegram Bot implementation**: a function to run the agent loop through Telegram Bot calling (i.e. through Telegram Bot chat)
